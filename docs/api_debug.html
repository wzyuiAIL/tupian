🔧 API调试工具

💡 使用说明：此工具用于诊断API调用问题，请按照步骤操作并查看日志输出。

1️⃣ API配置

⚠️ 注意：API地址应该以 /run 结尾，例如：https://px5r5j8mt8.coze.site/run
⚠️ 注意：Token应该以 pat_v2_ 开头（PAT格式）或 eyJ 开头（JWT格式），完整复制，不要有多余空格

2️⃣ 调试测试

📋 测试步骤：
  1. 点击"测试1：验证配置"检查API配置是否正确
  2. 点击"测试2：测试连接"检查能否连接到API服务器
  3. 点击"测试3：完整测试"模拟完整的API调用流程

4️⃣ 常见问题

❓ 如果出现"Failed to fetch"错误：
  • 检查API地址是否正确（包含https://和/run）
  • 检查Token是否正确（包含pat_v2_或eyJ前缀）
  • 检查网络连接
  • 检查浏览器控制台是否有CORS错误
  • 尝试使用不同的浏览器（Chrome/Edge）
