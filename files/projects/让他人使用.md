### 方式1：直接在线使用（无需下载）⭐⭐⭐

**最适合：快速体验、临时使用**

**步骤**：

1. 启用 GitHub Pages：

   - "Source" 选择 "Deploy from a branch"
   - "Branch" 选择 "main" → "Save"

2. 等待几分钟，访问：
   ```
   https://YOUR_USERNAME.github.io/ai-image-generator/
   ```

3. 使用在线CORS代理：
   - API地址：`https://corsproxy.io/?https://px5r5j8mt8.coze.site/run`
   - Token：用户的Coze API Token

✅ **其他人只需访问链接即可使用！**

---

### 方式2：下载使用（推荐）⭐⭐

**最适合：长期使用、稳定可靠**

**给用户的说明**：

#### 第1步：下载项目

1. 访问：https://github.com/YOUR_USERNAME/ai-image-generator
2. 点击 "Code" → "Download ZIP"
3. 解压到任意位置

#### 第2步：安装Python（如果没有）

1. 访问：https://www.python.org/downloads/
2. 下载并安装 Python 3.9+
3. 安装时勾选 "Add Python to PATH"

#### 第3步：启动代理服务器

**Windows用户**：
1. 进入 `scripts` 目录
2. 双击运行 `start_proxy_server.bat`

**Mac/Linux用户**：
```bash
cd scripts
pip install requests
python cors_proxy_server.py
```

#### 第4步：使用Web界面

1. 在浏览器中打开 `assets/web_sdk_index.html`
2. 配置API：
   - API地址：`http://localhost:8080/proxy`
   - Token：您的Coze API Token
3. 上传图片，生成！

## 🔧 高级配置

### 修改端口号

如果8080端口被占用：

1. 打开 `scripts/cors_proxy_server.py`
2. 找到最后一行：
   ```python
   def run_server(port=8080):
   ```
3. 修改为其他端口：
   ```python
   def run_server(port=9000):
   ```
4. 保存并重启

HTML页面的API地址也要改为：
```
http://localhost:9000/proxy
```

### 局域网访问

让同一网络的其他设备访问：

1. 查看本机IP：
   - Windows：`ipconfig`
   - Mac/Linux：`ifconfig`

2. 其他设备访问：
   ```
   http://你的IP:8080/proxy
   ```
