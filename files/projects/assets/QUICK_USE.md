# 📥 如何获取和使用HTML文件

## 🎯 快速开始（最简单的方式）

您的项目已经包含了完整的HTML文件，可以直接使用！

---

## 📍 文件位置

项目中有一个可以独立运行的HTML文件：

```
assets/web_sdk_index.html
```

**文件大小**: 26KB

---

## 🚀 如何获取文件

### 方法1：直接下载（如果您有项目访问权限）

1. 在文件管理器中找到项目目录
2. 进入 `assets` 文件夹
3. 找到 `web_sdk_index.html` 文件
4. 复制到您的电脑任意位置（桌面、文档等）

### 方法2：通过代码编辑器下载

如果您使用VS Code或其他编辑器：
1. 打开项目
2. 找到 `assets/web_sdk_index.html`
3. 右键点击文件
4. 选择"下载"或"另存为"

### 方法3：通过命令行

```bash
# 在项目根目录执行
cp assets/web_sdk_index.html ~/Desktop/ai_product_image_generator.html
```

---

## 📱 如何使用

### 第1步：打开HTML文件

找到下载的文件，双击用浏览器打开（Chrome、Edge、Safari等）

### 第2步：配置API

1. 在页面顶部找到"API配置"区域
2. 输入API地址（从Coze平台获取）
3. 输入API Token（从Coze平台获取）
4. 点击"保存配置"

### 第3步：上传图片

1. 点击第一个上传框，选择第1张产品图片
2. 点击第二个上传框，选择第2张产品图片
3. 支持格式：PNG、JPG、JPEG、GIF
4. 文件大小：最大16MB

### 第4步：选择风格

- 🤖 **智能自适应**（推荐）：AI自动分析
- 🔥 **暖色系**：温馨舒适
- ❄️ **冷色系**：现代清新
- ⚪ **中性色**：高端百搭

### 第5步：生成图片

1. 点击"生成图片"按钮
2. 等待2-5分钟
3. 查看生成的10张超高清图片
4. 点击"导出所有图片"批量下载

---

## 🔑 如何获取API地址和Token

1. 打开Coze平台
2. 找到"无水印产品图工作流"
3. 点击右上角"查看接入方式"
4. 复制API地址和Token
5. 在HTML页面中配置

**API地址格式示例**：
```
https://api.coze.com/v1/workflow/run
```

**Token格式示例**：
```
pat_xxxxxxxxxxxxxx
```

## 💡 使用技巧

### 图片上传建议
- ✅ 使用高清图片（至少1080p）
- ✅ 确保图片背景干净
- ✅ 两张图片最好角度一致
- ✅ 避免模糊或过度曝光的图片

### 风格选择建议
- **包包、鞋子**: 推荐使用"智能自适应"
- **服装**: 推荐使用"冷色系"
- **家居用品**: 推荐使用"暖色系"
- **通用**: 推荐使用"中性色"

### 生成时间
- 通常需要2-5分钟
- 如果图片较大，可能需要更长时间
- 请耐心等待，不要刷新页面

## ⚠️ 常见问题

### Q: 文件无法打开？
A: 确保用浏览器打开（Chrome、Edge、Safari等），不要用文本编辑器

### Q: API配置失败？
A:
- 检查API地址和Token是否正确
- 确保Token没有过期
- 尝试重新获取API信息

### Q: 上传图片报错？
A:
- 检查图片格式（只支持PNG/JPG/JPEG/GIF）
- 检查图片大小（最大16MB）
- 确保图片不是损坏的

### Q: 生成失败？
A:
- 检查API配置是否正确
- 检查网络连接
- 尝试重新生成
