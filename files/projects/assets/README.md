# 产品图片生成器 Web 界面

这是一个美观易用的Web界面，用于调用产品图片生成工作流。

## 📋 功能特性

- ✅ 图片上传和预览
- ✅ 智能风格检测（自适应）
- ✅ 三种风格选择（暖色系/冷色系/中性色）
- ✅ 一键生成10张高清图片
- ✅ 一键重新生成
- ✅ 批量导出图片
- ✅ 单张图片下载

## 🚀 使用方法

### 方式1：直接打开HTML文件

1. 在浏览器中打开 `index.html` 文件
2. 上传两张产品图片
3. 配置API Token（从部署页面获取）
4. 选择风格（可选，默认自适应）
5. 点击"生成图片"按钮

### 方式2：通过本地服务器（推荐）

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 访问
http://localhost:8000
```

### 方式3：部署到服务器

将整个 `assets` 文件夹上传到你的Web服务器即可。

## 🔑 获取API Token

1. 访问工作流部署页面
2. 点击 `[API Token]` 链接
3. 生成或复制你的Token
4. 在Web界面中输入Token

## 📸 支持的风格

### 🔥 暖色系
- 奶油色、米色、浅桃色
- 温馨、舒适、阳光氛围
- Z世代最爱，Instagram-worthy

### ❄️ 冷色系
- 天蓝色、薄荷绿、冰灰色
- 现代、清新、冷淡风
- 极简主义，高级感

### ⚪ 中性色
- 米白色、暖灰色、浅卡其色
- 高端、极简、百搭
- 经典不衰，适合任何品牌

### 🤖 自适应（推荐）
- AI自动分析图片
- 智能选择最佳风格
- 个性化定制

## 📊 输出结果

生成10张高清图片：
1. **主图**：2700×3600px（3:4）
2. **风格图1-8**：3200×3200px（1:1）
3. **详情页**：2700×3600px（3:4）

## 💡 使用技巧

1. **首次使用**：选择"自适应"风格，让AI自动判断
2. **需要特定风格**：直接选择暖色系、冷色系或中性色
3. **不满意结果**：点击"一键重新生成"
4. **批量导出**：点击"导出所有图片"按钮
5. **单张下载**：点击每张图片下方的"下载"按钮

## 🛠️ 技术栈

- **前端**：纯HTML + CSS + JavaScript
- **API**：Coze工作流API
- **无需构建**：可直接在浏览器中运行

## 📝 注意事项

1. API Token会保存在浏览器本地存储中
2. 生成的图片URL有效期为24小时
3. 建议及时下载保存重要图片
4. 每次生成都会创建新的图片

## 🎨 界面特点

- 响应式设计，支持各种屏幕尺寸
- 现代化UI，美观易用
- 实时预览上传的图片
- 显示智能风格检测结果
- 加载动画提示进度

## 📞 问题反馈

如果遇到问题，请检查：
1. API Token是否正确
2. 网络连接是否正常
3. 图片格式是否支持（JPG/PNG）
4. 图片大小是否合理（建议<10MB）
